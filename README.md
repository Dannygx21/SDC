
# SDC: System Design Capstone

A backend application in support of the Catwalk Application.


## Tech Stack

**Server:** Node, Express

**Database:** MongoDB

**ODM:** Mongoose

**Environment:** dotenv




## Features

- API supporting Catwalk webapplication
- ETL functionality to load MongoDB with CSV files


## In Development
- Server refactoring and optimization
- Database optimization and migration
- Docker containerization
